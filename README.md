# Threat Intelligence Tracker

This project documents emerging cybersecurity threats using open-source intelligence (OSINT).

The purpose of this repository is to practice structured threat research, document indicators of compromise, and organize findings in a clear and professional format.

## Project Structure

- `reports/` - threat reports and case studies
- `data/` - indicators of compromise and observable patterns
- `sources/` - research sources used for analysis
- `notes/` - analyst observations and investigation notes

## Skills Demonstrated

- threat intelligence research
- open-source intelligence (OSINT)
- security analysis
- structured documentation
- cyber threat tracking
